# 📊 Sales Performance Dashboard

## 📌 Overview

This project presents an interactive Sales Performance Dashboard developed using Power BI. The dashboard provides valuable insights into sales performance by tracking important business KPIs and visualizing sales trends.

---

## 🎯 Objectives

- Monitor Total Sales
- Analyze Total Profit
- Track Total Orders
- Monitor Quantity Sold
- Calculate Profit Margin
- Analyze Product Performance

---

## 🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- Power Query
- DAX

---

## 📈 Dashboard Features

- KPI Cards
- Monthly Sales Trend
- Product-wise Analysis
- Category-wise Analysis
- Interactive Dashboard

---

## 📊 KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Average Sales
- Profit Margin

---

- Data Cleaning
